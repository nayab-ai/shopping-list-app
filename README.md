# 🛒 Shopping List App

A simple and efficient **Shopping List Mobile Application** built using Flutter.  
This app helps users manage their daily shopping items with an easy-to-use interface and local storage support.

- ➕ Add new shopping items  
- 📝 Edit existing items  
- ❌ Delete items  
- 💾 Local storage using Hive Database  
- 📱 Clean and responsive UI  
- ⚡ Fast performance with Flutter

## 📦 Installation

1. Clone the repository:
```

git clone [https://github.com/nayab-ai/shopping-list-app.git](https://github.com/nayab-ai/shopping-list-app.git)

```

2. Move to project directory:
```

cd shopping-list-app

```

3. Install dependencies:
```

flutter pub get

```

4. Run the app:
```

flutter run

```
